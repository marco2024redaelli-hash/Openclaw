OpenClaw su Raspberry Pi 4

Guida interattiva — dal setup alla prima connessione Telegram

STEP 1 / 10 0%
1
Verifica Hardware
📋
Assicurati di avere tutto il necessario prima di iniziare.
🍓 Raspberry Pi 4 (4GB)
💾 microSD 16GB+
🔌 Alimentatore USB-C
🌐 Cavo Ethernet
📖 Lettore SD USB
🖥️ Cavo micro-HDMI
⚠️
Il cavo micro-HDMI è necessario per il debug. Se il Pi non si avvia correttamente, senza monitor non puoi vedere gli errori. Costo: ~7€ su Amazon.
  • ✓ Ho il Raspberry Pi 4 con la microSD inserita
  • ✓ Ho il caricatore USB-C (Moto G84 va bene)
  • ✓ Ho il cavo Ethernet e il router è vicino
  • ✓ Ho il lettore SD USB per il PC
2
Flash Raspberry Pi OS Lite
🔒
Usa Raspberry Pi Imager sul tuo PC per flashare il sistema operativo sulla microSD.
  • ✓ Apri Raspberry Pi Imager sul PC
  • ✓ Dispositivo → Raspberry Pi 4
  • ✓ OS → Raspberry Pi OS (other) → Raspberry Pi OS Lite (64-bit)
  • ✓ Archiviazione → la tua microSD
🚨
Deve dire "Lite" nel riepilogo! Senza "Lite" è la versione con desktop, troppo pesante per 16GB e OpenClaw.
3
Personalizzazione Imager
🔒
Configura le impostazioni prima di scrivere. Clicca su "Personalizzazione" nel menu di Imager.
  • ✓ Hostname: openclaw
  • ✓ SSH: Abilitato con password
  • ✓ Username: marco (o a scelta)
  • ✓ Password: scelta e memorizzata
  • ✓ Timezone: Europe/Rome
  • ✓ WiFi: saltato (usiamo Ethernet)
💡
Ricordati username e password — ti serviranno per collegarti via SSH!
4
Scrittura e Verifica
🔒
Clicca SCRIVI in Imager e aspetta. Il processo include scrittura + verifica e dura qualche minuto.
⏳
Non rimuovere la microSD durante la scrittura. Aspetta il messaggio "Scrittura completata con successo".
5
Collegamento del Pi
🔒
Collega tutto al Raspberry Pi nell'ordine corretto.
💻 PC HP ── WiFi ──→ 📡 Router Fastweb ←── Ethernet ── 🍓 Raspberry Pi

🍓 Pi ←── USB-C ── 🔌 Alimentatore (presa muro)
  • ✓ 1. Inserisci la microSD nel Pi (slot sotto la scheda)
  • ✓ 2. Collega il cavo Ethernet: Pi ↔ Router
  • ✓ 3. Collega alimentatore USB-C alla PRESA A MURO (non al PC!)
  • ✓ 4. Collega il cavo USB-C al Pi
  • ✓ 5. LED rosso fisso + LED verde lampeggia = OK
🚨
L'alimentazione va alla presa elettrica, NON alla porta USB del PC! Il PC non dà abbastanza corrente.
6
Connessione SSH
🔒
Aspetta 2 minuti per il primo avvio, poi collegati dal PC. Apri PowerShell (cerca nel menu Start).
ssh marco@openclaw.local
Se non funziona, prova con l'IP diretto. Cercalo sul router (192.168.1.254) o scansiona la rete:
ssh marco@192.168.1.XXX
Per trovare l'IP, scansiona la rete:
1..254 | % { $ip="192.168.1.$_"; if(Test-Connection $ip -Count 1 -Quiet -TimeoutSeconds 1){ Write-Host "$ip risponde" } }
💡
La prima volta ti chiede conferma — scrivi yes. Poi inserisci la password che hai scelto nello Step 3.
⚠️
Se hai timeout, collega un monitor micro-HDMI al Pi per vedere se il sistema si è avviato. Il LED verde che fa 4 lampeggi ripetuti = la microSD non viene letta.
7
Aggiornamento e Swap
🔒
Una volta dentro via SSH, aggiorna il sistema e crea lo swap (fondamentale con 4GB di RAM).
sudo apt update && sudo apt upgrade -y
sudo apt install -y git curl build-essential
Crea 2GB di swap:
sudo fallocate -l 2G /swapfile sudo chmod 600 /swapfile sudo mkswap /swapfile sudo swapon /swapfile echo '/swapfile none swap sw 0 0' | sudo tee -a /etc/fstab
Ottimizza per headless:
echo 'gpu_mem=16' | sudo tee -a /boot/config.txt sudo systemctl disable bluetooth sudo timedatectl set-timezone Europe/Rome
8
Installa Node.js 24
🔒
OpenClaw richiede Node.js 24. Installalo:
curl -fsSL https://deb.nodesource.com/setup_24.x | sudo -E bash - sudo apt install -y nodejs
Verifica:
node --version npm --version
💡
Deve mostrare v24.x.x per node. Se non è v24, ripeti il curl.
9
Installa OpenClaw
🔒
Installa OpenClaw con lo script ufficiale:
curl -fsSL https://openclaw.ai/install.sh | bash
Poi avvia l'onboarding:
openclaw onboard --install-daemon
Durante l'onboarding:
  • ✓ Homebrew → No (è per Mac)
  • ✓ Package manager → npm
  • ✓ Skills → Skip for now
  • ✓ API keys → No (configuri dopo)
  • ✓ Modello → anthropic/claude-sonnet-4-20250514
  • ✓ Canali → Skip for now (Telegram dopo)
🔑
Ti chiederà l'API key Anthropic. La trovi su console.anthropic.com → API Keys.
10
Connetti Telegram
🔒
Aggiungi Telegram come canale per controllare OpenClaw dal telefono.
openclaw channels add telegram
Poi su Telegram:
  • ✓ Cerca @BotFather su Telegram
  • ✓ Invia /newbot
  • ✓ Scegli un nome per il bot
  • ✓ Copia il token API fornito da BotFather
  • ✓ Incolla il token nel terminale SSH
  • ✓ Cerca il tuo bot su Telegram e invia /start
🎉
Fatto! Ora puoi parlare con OpenClaw direttamente da Telegram sul tuo Moto G84. Il Pi resta acceso 24/7 e OpenClaw è sempre disponibile!
